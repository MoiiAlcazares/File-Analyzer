File Analyzer PRO

File Analyzer PRO is a Python-based command-line tool that analyzes text files and generates structured reports.

---

## 🚀 Features

- Reads `.txt` files
- Counts total lines, words, and characters
- Cleans text by removing punctuation and common stopwords
- Identifies the most frequent words
- Generates a detailed analysis report
- Exports results to CSV
- Optional Excel export support

---

## 🧠 What this project demonstrates

This project was built to practice and demonstrate:

- File handling in Python
- Text processing and data cleaning
- Basic data analysis concepts
- Automation of report generation
- Working with structured output formats (CSV / Excel)

---

## 📁 Output Files

The program generates:

- `reporte.csv` → structured analysis in CSV format  
- `reporte.xlsx` → Excel report (optional, if enabled)
